# 6-Layered Fuel Cell Monitoring ESP32

A six-layer fuel cell monitoring PCB designed in KiCad for accurate voltage and current measurement. The system integrates analog signal conditioning, precision ADC interfaces, and an ESP32-S3 microcontroller for data acquisition and processing.

## Features

* 6-layer PCB design
* Fuel cell voltage monitoring
* Fuel cell current monitoring
* ESP32-S3 integration
* Custom KiCad symbols and footprints
* 3D PCB model

## Tools

* KiCad
* ESP32-S3
* Git & GitHub
